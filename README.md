## Description

Simple project for trying [Nest](https://github.com/nestjs/nest) framework for building rest apis
and [Prisma](https://www.prisma.io/) for database/ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
